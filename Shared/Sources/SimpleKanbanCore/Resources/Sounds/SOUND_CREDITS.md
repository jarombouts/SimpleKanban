## Download Instructions

1. Go to each URL above
2. Click "Download" (requires free Freesound account)
3. Rename file to match the names above (e.g., `sad_trombone.mp3`)
4. Convert to MP3 if needed (some are WAV)
5. Normalize audio to -1dB peak
6. Trim silence from start/end
7. Place in `Shared/Resources/Sounds/`

## Audio Specs Target

- Format: MP3 (or WAV for short UI sounds)
- Sample Rate: 44.1kHz
- Bit Rate: 128-192 kbps
- Duration: 0.5 - 3 seconds (except chant)
- Channels: Stereo or Mono
